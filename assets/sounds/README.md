# أصوات الإشعارات

يجب إضافة الملفات الصوتية التالية بصيغة .mp3:

## الأصوات المطلوبة:
- `message.mp3` - صوت الرسالة الجديدة
- `group.mp3` - صوت رسالة المجموعة
- `call.mp3` - صوت المكالمة الواردة
- `notification.mp3` - صوت الإشعار العام
- `sent.mp3` - صوت إرسال الرسالة
- `camera.mp3` - صوت التقاط الصورة
- `recording_start.mp3` - صوت بدء التسجيل
- `recording_end.mp3` - صوت انتهاء التسجيل

## كيفية الحصول على الأصوات:
1. استخدم مواقع الأصوات المجانية مثل:
   - https://freesound.org
   - https://mixkit.co/free-sound-effects
   
2. أو استخدم أصوات Telegram المفتوحة المصدر

3. تأكد من أن الأصوات قصيرة (0.5-2 ثانية) وبجودة جيدة
